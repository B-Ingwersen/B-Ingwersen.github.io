Grapher 3D

A basic graphic calculator than can display two and three-dimensional equations.

After using products like Desmos, I was interested trying to make my own graphing program, especially as my Multivariable Calculus class was annoyed by the lack of built in 3D graphing in Desmos. Obviously, this project is nowhere near as developed as the programs it was inspired by, but it has support for basic two-dimensional and three-dimensional graphs. Additionally, it is very responsive as it doesn't have the bloat of large computer algebra systems. The screenshots below show some of the fun graphs that the program can produce.
