Graphics Library

A lightweight software rendering library that can render to windows with SDL or directly to the Linux Frame Buffer.

I've found graphics to be a very interesting area, and I spent some time trying to find a platform where I had pixel level control with good performance. Over time, this evolved into a small software rendering library that has been compiled into this project. It now provides basic GUI elements and some support for rendering images and fonts. The library mainly uses SDL2 for connecting to the window manager. On Linux, the project can also render directly to the framebuffer, allowing unmodified applications to run graphically when launched from a tty. I've made several smaller projects that use this library as their graphical base (including Grapher 3D) which can be seen in the screenshots below.

Quick Information

  • Language: C++
  • Project Size: ~8,000 Source Lines of Code
  • Extensions for Rendering: PNG, JPG, TrueType Fonts
  • Compiled Library Size: ~200 kb
