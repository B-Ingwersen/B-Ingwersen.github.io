Ray Tracer

A simple ray tracer capable of rendering reflections and simulating depth of field.

This hobby project started as a curiosity to see what 3D rendering I could do with my basic knowledge of geometry and physics. The program has a few basic object types built in that can be rendered, but there is an interface for adding your own custom 3D objects. Recently, I reorganized the code and made the math a lot more elogant as I knew very little linear algebra when I began writing it.
